## 사용자
- 평소에 우리 동네 미세먼지 수치를 알고 싶은 사람들
- 호흡기 질환이 있어 미세먼지 수치에 예민한 사람


## 기능
### 미세먼지 측정기능
- 현재 위치를 측정하여 가장 가까운 대기 오염 관측소 정보를 받아옵니다.
- 가장 가까운 대기 오염 관측소의 미세먼지 정보를 보여줍니다.

### 위젯 기능
- 미세먼지 수치를 앱을 진입하지 않고도 볼 수 있도록 하는 기능입니다.
